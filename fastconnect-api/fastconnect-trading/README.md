# FastConnect Trading

FastConnect Trading bao gồm:&#x20;

\+ FCTrading API: Bộ API tương tác với hệ thống giao dịch

\+ FCTrading Streaming: Luồng trả về trạng thái lệnh và thông tin cập nhật cho Khách hàng.

Tất cả sẽ được lưu trữ tại SSI.
